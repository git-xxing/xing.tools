# JDK安装与部署
<CodeGroup>
<CodeGroupItem title="ubuntu" active>

```shell
apt install openjdk-8-jdk
java -version
```
</CodeGroupItem>
<CodeGroupItem title="yum">

```shell
yum list java*
yum -y install java-1.8.0-openjdk*
java -version
```
