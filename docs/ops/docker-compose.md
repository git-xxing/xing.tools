# Docker Compose
## 安装
- 安装前请先安装 [Docker](docker.md)

<CodeGroup>
<CodeGroupItem title="yum" active>

```shell
 sudo yum update
 sudo yum install docker-compose-plugin
```

</CodeGroupItem>
<CodeGrpupItem title="apt-get" >

```shell
 sudo apt-get update
 sudo apt-get install docker-compose-plugin
```

</CodeGrpupItem>
</CodeGroup>

- 查看版本号
```shell
 docker compose version
```
